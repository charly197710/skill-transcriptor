# 🎤 Meeting Transcriber

Aplicación para transcribir y analizar audio/video.

## 📦 Descarga directa

Descarga `MeetingTranscriber.exe` desde releases - ¡no requiere instalación!

## ▶️ Uso

### Escritorio
```bash
run.bat
```

### Web (nube)
```bash
uv run python main.py --web
```
